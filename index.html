Importing Excel List into
Word Dropdown List

My job is to create letters from a stored template to send out to customers who have requested information on people.
Often, we receive these requests from the same person or company. I wondered if there was a way I didn't have to type out the name and address of customers every time they sent us a request. I spoke with my boss about this, and he liked the idea and allowed me to work on the project. So, with my very limited coding knowledge but very good Google searching skills, I researched how to use Microsoft Word's Visual Basic Application and I found a suitable code for my template. The code that I found had several different options for drop down lists, so I had to whittle it down so that the code only included the sections that I needed.
Suffice to say, there may be strings of code that are irrelevant for which I can only apologise as my knowledge of computer science is only at a beginner level.
